# Azure-Zero-Trust-Lab
Azure Zero Trust Landing Zone built with Terraform. Hub-spoke network with Azure Firewall, Bastion (no public IPs), NSGs, Key Vault with RBAC, Managed Identity, Log Analytics, and Azure Policy enforcement. Implements least-privilege access, network segmentation, and centralized logging. Full IaC for secure Azure workloads.
